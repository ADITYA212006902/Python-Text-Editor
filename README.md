# 📝 Simple Text Editor (Notepad Clone)

A lightweight and user-friendly **Notepad application** built using **Python Tkinter GUI**. This project allows users to create, open, edit, and save text files with ease.

---

## 🚀 Features

* 📄 Create a new file
* 📂 Open existing `.txt` files
* 💾 Save files
* 🖥️ Clean and simple GUI
* 🔤 Word wrapping enabled
* ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI Library)

## ▶️ How to Run

1. Install Python (if not already installed)
2. Clone this repository:

   ```bash
   git clone https://github.com/your-username/simple-text-editor.git
   ```
3. Navigate to the project folder:

   ```bash
   cd simple-text-editor
   ```
4. Run the program:

   ```bash
   python your_file_name.py
   ```
